# WanAndroid 

# 1.功能及特色
RxJava + Retrofit2 + okHttp3 + MVP实现的wanAndroid客户端。<br>
主要功能：<br>
1. Android体系和网站导航<br>
2. 全局收藏和取消收藏，可在我的收藏中查看<br>
3. 热门搜索和历史搜索

在此特别感谢[玩Android开放API](https://www.wanandroid.com/blog/show/2)
